Singling 1.0.5 for 1.07 - 1.13d
By Jonathan Vasquez (fearedbliss)
Released on Tuesday, June 10, 2014
------------------------------------------------

This is a light addon to improve the Single Player experience. The below features are modifications
that are accepted by the Diablo IncGamers Single Player Forum (SPF).

Overall this serves as a consolidation of FAM (Forum Accepted Mods).

This bundle supports the following versions: 1.07, 1.08, 1.09d, 1.10f, 1.11b, 1.12a, 1.13d

1.07 - 1.10f
-----------------
- You can now run multiple clients of Diablo II.

1.11b - 1.13d
-----------------
- Ladder Runewords are now unlocked and can be used on Single Player.

- Runes are now colored orange

- You can now run multiple clients of Diablo II.
