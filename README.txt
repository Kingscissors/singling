Singling 1.0.3 for 1.07 - 1.13d
By Jonathan Vasquez (FearedBliss)
Released on Monday, September 30, 2013
------------------------------------------------

This is a light addon to improve the Single Player experience. The below features are just things that the
Single Player Forum (SPF) at Diablo IncGamers already accept.

Overall this serves as a consolidation of FAM (Forum Accepted Mods).

This bundle supports all versions from 1.07 to 1.13d (One version of each patch was selected):
1.07, 1.08, 1.09d, 1.10f, 1.11b, 1.12a, 1.13d.

1.07 - 1.09d
-----------------
- Runes are now colored orange

- You can now run multiple clients of Diablo II.

1.10f - 1.13d
-----------------
- Ladder Runewords are now unlocked and can be used on Single Player.

- Runes are now colored orange

- You can now run multiple clients of Diablo II.
